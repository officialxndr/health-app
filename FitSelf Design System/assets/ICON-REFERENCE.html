FitSelf — Icon Reference

Every icon in FitSelf is lucide, re-exported from one file (apps/web/src/components/icons.ts). Stroke-based, rounded, even weight. Default stroke 2; active states 2.4–2.5. No emoji, no unicode glyphs, no custom icon set. Pull from CDN to match exactly.
